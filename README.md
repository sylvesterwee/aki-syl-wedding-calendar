# Aki & Syl's Wedding in Tokyo 🇯🇵 🌸 Calendar Link

This is a tiny static GitHub Pages site for sharing one calendar link over WhatsApp.

## What guests see

Guests open the GitHub Pages URL and choose one of:

- Google Calendar
- Outlook Calendar
- Apple Calendar
- Download `.ics`

## Files

- `index.html` - the calendar chooser page
- `aki-syl-wedding.ics` - calendar invite used by Apple Calendar and other apps
- `.nojekyll` - tells GitHub Pages to serve files as-is

## Publish With GitHub Pages

1. Create a new public GitHub repository.
   Suggested name: `aki-syl-wedding-calendar`

2. Upload these files to the repository:
   - `index.html`
   - `aki-syl-wedding.ics`
   - `.nojekyll`
   - `README.md`

3. In GitHub, go to:
   `Settings` -> `Pages`

4. Under `Build and deployment`, choose:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/ (root)`

5. Save. GitHub will publish the site after a minute or two.

Your final link will look like:

`https://YOUR-GITHUB-USERNAME.github.io/aki-syl-wedding-calendar/`

Paste that link into `whatsapp-calendar-message.txt`.
