# Free Single-Link Calendar Page

This folder contains a no-paid-platform calendar chooser:

- `index.html` - the page guests open
- `aki-syl-wedding.ics` - the Apple Calendar / Outlook / generic calendar invite
- `whatsapp-calendar-message.txt` - a short WhatsApp message once you have the final URL

## Free hosting options

Use any free static hosting provider. The easiest choices are:

1. GitHub Pages
2. Netlify Drop
3. Cloudflare Pages

Upload or publish both `index.html` and `aki-syl-wedding.ics` in the same folder. The single link to send is the hosted `index.html` page.

## WhatsApp message

After hosting, replace `PASTE_YOUR_FREE_HOSTED_LINK_HERE` in `whatsapp-calendar-message.txt` with your final page URL.
